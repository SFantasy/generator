Generator
---

**PACKAGE DEVELOPING**

## Install

```
[sudo] npm install npm-generator [-g]
```

## Usage

### Generate package

```
npm-generator test
```

### Sample structure

```
|--bin
| |- pkg
|--lib
| |- index.js
|--test
| |- test.js
|-index.js
|-package.json
|-README.md
|-LICENSE
|-.travis.yml
|-.npmignore
|-.gitignore
```
